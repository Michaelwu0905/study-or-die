# study-or-die

一个模仿 cowsay / fortune 风格的终端学习激励工具。  
每隔指定秒数输出一条嘲讽语录，并配合用户自定义的 ASCII 角色进行展示。  
支持多角色轮流输出、运行时间与输出频率自定义。

---

## 功能特性

- 从 `character.txt` 中读取多个角色（空行分隔）
- 从 `quotes.txt` 中读取多条语录
- 轮流输出角色与语录（非随机）
- 支持用户通过参数设置运行时长与输出频率
- 支持随时按 `q` + 回车退出程序

---

## 使用方法

### 基本运行

进入项目根目录，然后运行

```bash
# 默认运行 5 分钟，每 60 秒输出一次
./install.sh

study-or-die -t <运行时间(分钟)> -f <运行频率(秒)>
```
